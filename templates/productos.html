¿Qué es SQLAlchemy ORM?

SQLAlchemy ORM permite trabajar con la base de datos usando objetos Python en lugar de escribir SQL directamente. La clase ProductoORM mapea la tabla productos_orm. Las operaciones CRUD se realizan con db.session.add(), db.session.delete() y db.session.commit().
