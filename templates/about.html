SmartShop

Sistema de Gestión de Inventario — 2026

Tecnologías utilizadas
  • Flask — Framework web Python
  • SQLite — Base de datos embebida (inventario principal)
  • SQLAlchemy ORM — Mapeo objeto-relacional
  • MySQL — Base de datos relacional (módulo avanzado)
  • Archivos — Persistencia en TXT, JSON y CSV
  • ReportLab — Generación de reportes PDF
  • Bootstrap 5 — Interfaz responsiva
Arquitectura del proyecto
