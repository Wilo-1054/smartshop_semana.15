Cómo funciona la conexión MySQL

El módulo conexion/conexion.py centraliza los parámetros de conexión (host, user, password, database). La función get_connection() usa mysql-connector-python para abrir una conexión; close_connection() la cierra de forma segura.

Todas las consultas usan parámetros posicionales (%s) para prevenir inyección SQL. El diccionario cursor=conn.cursor(dictionary=True) retorna filas como dict para facilitar el uso en las plantillas Jinja2.
